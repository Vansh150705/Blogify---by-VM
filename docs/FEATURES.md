# Features

A closer look at what Blogify offers.

## Public site
- Browse published blog posts on the home page
- Read a full blog post with rich formatting
- Leave comments on posts (subject to admin approval)
- Responsive layout for mobile, tablet, and desktop

## Admin dashboard
- Secure JWT-based login
- Create, edit, publish/unpublish, and delete posts
- Rich-text editor (Quill) with image uploads
- View and moderate comments (approve / delete)
- Dashboard overview with counts and recent activity

## AI & media
- **AI content generation** via Google Gemini to draft or expand post content
- **Optimized image hosting** via ImageKit for fast, transformed images

## Developer experience
- Conventional commits and CI (lint + build) on every push
- EditorConfig and Prettier for consistent formatting
- Documented REST API and deployment guide under `docs/`
