# Roadmap

A living list of planned improvements for Blogify. Items are aspirational and
subject to change.

## Near term
- [ ] Pagination for the blog list
- [ ] Search and tag filtering on the home page
- [ ] Loading skeletons for blog cards
- [ ] Form validation feedback on the admin editor

## Mid term
- [ ] Draft auto-save in the editor
- [ ] Rich comment moderation (spam flagging)
- [ ] Server-side input validation with a schema library
- [ ] Unit tests for controllers

## Long term
- [ ] Multi-author support with roles
- [ ] Email notifications for new comments
- [ ] Analytics dashboard (views, engagement)
- [ ] Dark mode

Have an idea? Open a feature request issue.
