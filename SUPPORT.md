# Support

Thanks for using Blogify! Here's how to get help.

## Documentation

Start with the docs before opening an issue:

- [README](./README.md) — setup and overview
- [docs/API.md](./docs/API.md) — REST API reference
- [docs/DEPLOYMENT.md](./docs/DEPLOYMENT.md) — deploying to Vercel
- [docs/TROUBLESHOOTING.md](./docs/TROUBLESHOOTING.md) — common problems
- [docs/FAQ.md](./docs/FAQ.md) — frequently asked questions

## Asking a question

- For usage questions, open a GitHub Discussion.
- For bugs, open an issue with the **Bug report** template.
- For ideas, open an issue with the **Feature request** template.

## Security

Please report vulnerabilities privately — see [SECURITY.md](./SECURITY.md).
